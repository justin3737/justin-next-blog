> 每次學習新東西都是寫 Todo list 實在太無趣了，這是我個人 redux 學習紀錄：使用 react 與 redux 開發一個留言板，並且把相關程式碼放在 github 上供參考

留言列表的 user_id 與留言者的 user_id 相同

- modals user_id === lists user_id

- ex: ‘JUSTIN’ === ‘JUSTIN’

- user_id 由 user 的字串轉大寫: toUpperCase()

Reducers (Store)的設計：

```js showLineNumbers
modals:
{
  visible:false,
  message:"", /*輸入時暫存的內容*/
  user:"Justin",
  user_id:"JUSTIN"
}

lists:
[{
  id:3
  message:""  /*最後儲存的內容*/
  user:"Justin"
  user_id:"JUSTIN"
  time:"2016-3-8 10:50:38"
}]
```

- 開啟 ./src/components/ListItems.js

- 這裡透過 css classname 來決定是否要顯示 [x] 按鈕

```js showLineNumbers
_setClassName = () => {
  const { user_id, curr_user } = this.props
  return classNames('itemWrap', user_id === curr_user ? 'is-my-msg' : '')
}
```
